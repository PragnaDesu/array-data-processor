Array Data Processor

A REST API that processes input arrays and returns categorized results including numbers, alphabets, special characters, sums, and transformed strings. Built to demonstrate API design, array manipulation, and frontend-backend integration.

Smart Categorization

Separates numbers, alphabets, and special characters

Mathematical Processing

Calculates sums and identifies odd/even numbers

String Transformation

Creates concatenated strings with intelligent sorting

API Specification

The Array Data Processor API accepts POST requests with JSON array data and returns structured results.

Endpoint

POST /process

Request Format

{ "data": ["a", "1", "23", "$", "B"] }

Response Format

{ "is_success": true, "user_id": "demo_user_29082025", "email": "demo@example.com", "roll_number": "12345", "odd_numbers": ["1", "23"], "even_numbers": [], "alphabets": ["B", "a"], "special_characters": ["$"], "sum": "24", "concat_string": "Ba" }

Field Descriptions

  • odd_numbers: Array of odd numbers as strings
  • even_numbers: Array of even numbers as strings
  • alphabets: Array of alphabetic characters (uppercase first)
  • special_characters: Array of non-alphanumeric characters
  • sum: Sum of all numbers as string
  • concat_string: Concatenated alphabets (uppercase first, then lowercase)
